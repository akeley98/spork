\magicSubsection{Sync Check}{sec:gSyncCheck}

Checks that a program does not reach an error condition under abstract machine semantics (Section~\ref{sec:SyncSemantics}).
This is intended to ensure sequential-parallel equivalence (def~\ref{sec:gSeqParEquivalence}), although we don't prove this at all.
Use the \lighttt{exo.Procedure.sync\_check} function with the problem size given as keyword arguments, e.g. \lighttt{my\_gemm.sync\_check(M=512, N=256, K=384)}.
