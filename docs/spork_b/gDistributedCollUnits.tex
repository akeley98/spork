\magicSubsection{Distributed Collective Units}{sec:gDistributedCollUnits}

Per-instruction parameter list of $N$-many collective units (def~\ref{sec:gCollUnit}).
This specifies that the first $N$ dimensions of the parameter are distributed dimensions (def~\ref{sec:gDistributedDimension}), while the others are not.
The interpretation of this is specified in Section~\ref{sec:InstrDistributedMemory}.
