\magicSubsection{wgmma Fence}{sec:gWgmmaFence}

Fence of the form \texttt{Fence(wgmma\_fence\_1, wgmma\_fence\_2)} at \lighttt{cuda\_warpgroup}-scope (def~\ref{sec:gCollUnit}).
A thread must execute this before any usage of registers in \lighttt{wgmma} instructions.
See also ``Fence Usage'' (Section~\ref{sec:FenceUsage}).
