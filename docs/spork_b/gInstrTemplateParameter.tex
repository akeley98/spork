\magicSubsection{Instruction Template Parameter}{sec:gInstrTemplateParameter}

Parameter named in the \lighttt{instance} function of an instruction template (Section~\ref{sec:InstrTemplate}), except for the left-most argument (\lighttt{self}).
Any parameters in the \lighttt{behavior} function whose name matches that of an instruction template parameter (as defined by \lighttt{instance}) is itself an instruction template parameter.
