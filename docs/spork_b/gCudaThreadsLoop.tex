\magicSubsection{\texttt{cuda\_threads} Loop}{sec:gCudaThreadsLoop}

Loop with loop mode \lighttt{CudaThreads} (def~\ref{sec:gLoopMode}).
Such loops subdivide their executing thread collective (def~\ref{sec:gThreadCollective}) into subsets, each assigned to execute an ``iteration'' of the loop in parallel.
See Section~\ref{sec:CollTilingCudaThreads}.
