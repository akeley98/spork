\magicSubsection{First Synchronization Timeline}{sec:gFirstSyncTL}

First synchronization timeline (def~\ref{sec:gSyncTL}) parameter of a \lighttt{Fence} statement, or only synchronization timeline parameter of an \lighttt{Arrive} statement.
Denoted $\tau_s^\text{pre}$.
This filters by qualitative timeline (def~\ref{sec:gQualTL}) the memory accesses prior to the synchronization statement (def~\ref{sec:gSyncStmt}) that interact with the synchronization (Section~\ref{sec:AccessBeforeSync}).
