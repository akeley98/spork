\magicSubsection{\texttt{CudaWarps} Block}{sec:gCudaWarpsBlock}

Warp specialization statement of the form ``\lighttt{with CudaWarps(...): $s$}'', which restricts instances of the child statement $s$ to execute using only a certain subset of warps.
These warps may be identified by a warp variable (def~\ref{sec:gWarpVariable}).
See Section~\ref{sec:CollTilingCudaWarps}.
