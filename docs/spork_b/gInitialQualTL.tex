\magicSubsection{Initial Qualitative Timeline}{sec:gInitialQualTL}

Qualitative timeline (def~\ref{sec:gQualTL}) associated with each read/mutate memory access.
This varies depending on the memory type accessed and the instruction used (Section~\ref{sec:InstrTL}).
In abstract machine semantics, this controls \textsf{VisRecord} creation (Section~\ref{sec:VisRecordCreation}).
