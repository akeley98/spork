\magicSubsection{Second Synchronization Timeline}{sec:gSecondSyncTL}

Second synchronization timeline (def~\ref{sec:gSyncTL}) parameter of a \lighttt{Fence} statement, or only synchronization timeline parameter of an \lighttt{Await} statement.
Denoted $\tau_s^\text{post}$.
This filters by qualitative timeline (def~\ref{sec:gQualTL}) the memory accesses subsequent to the synchronization statement (def~\ref{sec:gSyncStmt}) that interact with the synchronization (Section~\ref{sec:AccessAfterSync}).
