\magicSubsection{Extended Qualitative Timelines (Set)}{sec:gExtQualTL}

Set of qualitative timelines (def~\ref{sec:gQualTL}) associated with each read/mutate memory access.
This varies depending on the memory type accessed and the instruction used (Section~\ref{sec:InstrTL}).
In abstract machine semantics, this controls the checks upon reading (Section~\ref{sec:ChecksOnRead}) or mutating (Section~\ref{sec:ChecksOnMutate}) an array element.
